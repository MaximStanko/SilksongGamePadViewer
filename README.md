# Silksong GamePadViewer

## General

Hollow Knight: Silksong themed GamePadViewer skin. GamePadViewer enables the display of controller inputs as an OBS overlay. This is a fork of https://github.com/AlexanderSkrock/GamePadViewer

## Supported Controller types

Currently only xbox one controllers are supported. For any other controllers it would be neccessary to change the base images and positioning. This might be part of a future change.

## How to use

Use the following as custom CSS in the GamePadViewer website:
